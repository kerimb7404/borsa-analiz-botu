# 📈 Finans Takip Dashboard

Bu proje, hem kripto paraları hem de BIST (Borsa İstanbul) hisselerini tek bir ekrandan takip etmeni sağlayan modern ve hızlı bir dashboard uygulamasıdır. 

## ✨ Özellikler
* **Canlı Veri:** Kripto para fiyatları API üzerinden anlık çekilir.
* **Görsel Grafikler:** Chart.js kullanılarak oluşturulan interaktif fiyat geçmişi.
* **Hibrit Takip:** Hem Crypto hem de BIST verilerini aynı anda görebilme.
* **Modern Arayüz:** Karanlık mod (Dark Mode) uyumlu, hızlı ve kullanıcı dostu tasarım.
* **Mobil Uyumlu:** Telefonlardan kolayca erişilebilir.

## 🛠️ Kullanılan Teknolojiler
* **HTML5 & CSS3:** Temiz ve responsive tasarım.
* **JavaScript (Vanilla):** Veri yönetimi ve API entegrasyonu.
* **Chart.js:** Verilerin görselleştirilmesi.

## 🚀 Nasıl Çalıştırılır?
1. Bu repoyu bilgisayarınıza indirin veya klonlayın.
2. `index.html` dosyasını herhangi bir modern tarayıcıda açın.
3. Hepsi bu! Kurulum gerektirmez.
